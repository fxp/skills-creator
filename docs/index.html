How It Works

The expert talks. The agent asks probing questions, generates a skill, tests it, and exports to your framework of choice.

🎙 Voice-First Design

One question per turn. Verbal summaries at transitions. Optimized for speech, not text walls.

🧠 Deep Knowledge Extraction

Socratic questioning probes for edge cases, failure modes, quality criteria, and domain vocabulary.

🧪 Closed-Loop Testing

Auto-generates behavioral evals from conversation examples. Runs tests. Iterates until they pass.

📦 Multi-Framework Export

One conversation → skills for Claude Code, OpenClaw, Cursor, Hermes, or any LLM.

Quick Start

git clone https://github.com/xiaopingfeng/skills-creator.git cd skills-creator && pip install -r requirements.txt # Register as OpenClaw agent openclaw agents add skills-creator --workspace ./workspace # Set your LLM API key export OPENAI_API_KEY=your-key export OPENAI_BASE_URL=https://your-api/v1 # Start openclaw gateway && openclaw tui
